# 📝 Gestor de Tareas en Java

Aplicación de consola desarrollada en Java para gestionar tareas utilizando Programación Orientada a Objetos y estructuras dinámicas.

---

## ⚙️ Funcionalidades

- Añadir tareas
- Ver lista de tareas
- Marcar tareas como completadas
- Eliminar tareas
- Identificación de tareas mediante ID único.

---

## 🛠️ Tecnologías utilizadas

- Java
- Eclipse IDE
- GitHub

---

## 📁 Estructura del proyecto

```
src/
└── gestiontareas/
    ├── Tarea.java
    └── GestionTareas.java
```

---

## ▶️ Cómo ejecutar el proyecto

1. Abrir el proyecto en Eclipse.
2. Ejecutar la clase `GestionTareas`.
3. Usar el menú de consola para interactuar.
